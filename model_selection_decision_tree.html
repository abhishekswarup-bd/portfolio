🏆 Selection Criteria

1

Free

Must run on free API tiers. Zero monthly cost. Sustainable for ongoing use without budget constraints.

2

Different Architectures

Each model must be a distinct "brain" â different training data, different biases, different failure modes.

3

Different Strengths

Reasoning / Sentiment / Quant / Validation â each model fills a unique, non-overlapping role in the pipeline.

4

API-Callable

Must be accessible via API. No self-hosting, no GPU provisioning, no infrastructure overhead required.

5

No Downgrades

Reject any model smaller or weaker than what's already in the pipeline. Every addition must raise the bar.

6

Cascading Validation

Each model sees prior outputs. Gemini cross-checks all three. Disagreements trigger human review.
